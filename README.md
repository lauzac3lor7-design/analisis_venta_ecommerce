# análisis_venta_ecommerce
Análisis de datos de una tienda en línea global utilizando SQL para la limpieza y Python para el cálculo de métricas de ventas y visualización
En este proyecto se busca analizar el comportamiento de ventas de una tienda en linea internacional de regalos del Reino Unido.El objetivo fue encontrar los factores que impulsan la rentabilidad de este negocio y encontrar maneras de que siga funcionando e impmementar nuevas ideas que sigan impulsando su crecimiento basandose en sus datos reales.

### Herramientas Utilizadas:
* **Python:** Limpieza de Datos, manejo de fechas y visualización
* **SQL:** Consultas para extracción de métricas
* Google Colab: Entorno de ejecución y desarrollo para el análisis.

### Hallazgos principales:
*La Cantidad (Quantity) tiene una correlación del 85% con el éxito del monto total de venta.
* Existe una correlación negativa del -15% con el precio, lo que indica que el negocio depende de productos económicos más que de artículos de lujo.
* El White Hanging Heart T-Light Holder lidera las ventas con más de 11,000 unidades, confirmando la estrategia de volumen masivo.

### Conclusión:
De acuerdo a todo el análisis hecho , la cantidad es mucho más importante que el precio o el horario de venta para mover el monto total de facturación y es la que impulsa el crecimiento. A medida que el precio unitario sube, el monto total de la venta tiende a bajar ligeramente. Probablemente se venden muchos artículos de bajo costo pero en cantidades masivas.Cuando se intenta vender productos más caros, el volumen cae tanto que el ingreso total no compensa esa subida de precio. Para maximizar los ingresos se debe enfocar en capturar oportunidades perdidas, es decir los de alamacén y despacho se deben concentrar en hacer entregas rápidas.Los dias sábados en donde ocurren menos ventas es una oportunidad para dar ofertas sin saturas los días cargados.Por la alta correlación de cantidad, la operación debe optimizarse para incentivar compras de mayor volumen en lugar de solo subir precios.
